Python

13 Oct, 2020 18 Nov, 2020

Python is a language designed to develop software easy and practical. Due to its low learning cost and high productivity, it is now used in various fields from web application construction to data analysis, and in recent years it has become a very popular language among many programming languages. This page summarizes the features of Python.

Used for developing famous services

  • Youtube
  • Instagram
  • Dropbox

These services are developed using Python. In recent years, Python has also been active in machine learning in the scientific field and in the development of AI as well.

High readability

Python requires less program codes to write than C or Java, which is making it easier to program. This is because the grammar itself is simple and designed so that anyone can write the same code. For example, if you write a program that calculates the absolute value of a number, you can see the difference by comparing Python and Java.

Python
def absolute(x):
    if x >= 0:
        return x
    else:
        return -x
Java
int absolute(int x) {
    if (x >= 0) {
        return x;
    } else {
        return -x;
    }
}

However, in Java, writing in this way is also correct and does not cause an error.

int absolute(int x) {
if (x >= 0) {return x;} else {
        return -x;}}

In Python, indentation is determined by syntax rules (an error will occur if you do not follow it), so you cannot write it as you like in Java. This may seem like a disadvantage, but it simplifies the syntax and makes it easier to read, makes it harder for people to write code differently, and that makes it easier to read code written by another person. This has helped reducing costs of learning Python, reducing development time, and increasing productivity.

Can be operated in various environments

Python can be run in the following environment:

  1. Linux
  2. Windows
  3. MacOS
  4. iOS
  5. Android

Can be run in various environments means that it is highly versatile and can be used for various purposes.

Summary

  • A language that has grown in popularity in recent years
  • Can develop web applications, machine learning, and AI
  • Beginner friendly

If you are interested, download Python from the official website and start studying.
