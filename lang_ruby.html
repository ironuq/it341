Ruby

13 Oct, 2020 18 Nov, 2020

Ruby is one of the most popular programming languages today. It's not only beginner-friendly, but also so many services are developed in Ruby, so once you learn, you can use it for your job too. This page summarizes some features of Ruby.

Easy to learn

Among many programming languages, Ruby has a simple grammar and is easy for beginners to learn. Ruby is also a language developed with the aim of making programmers enjoyable coding, as stated in "Enjoy Programming". Specifically, compared to other programming languages, programming-specific symbols are minimized, and programs can be written with a smaller number of characters.

Large number of "packages"

Ruby is a versatile programming language that allows you to create various services mainly web applications. The reason for its versatility is its over 8000 of packages. Package is something that can implement various functions with less coding. You can implement all the functions on the web with short code by using different packages for each function you want to realize.

Ruby on Rails

Ruby has a framework called Ruby on Rails, and by using it, you can develop various web applications and services very efficiently. This high development efficiency is the main reason of Ruby's popularity. With its simplicity of grammar, the framework Ruby on Rails, and over 8000 packages, Ruby is not only easy to read and write, but it also allows us to efficiently develop any web application service.

What you can do with Ruby

Web application development is the area where Ruby is most good at and used. For example, suppose you want to develop an online shopping website. By using Ruby and Ruby on Rails, you can add the following functions required for online shopping sites.

  1. Search
  2. Order
  3. Online payment
  4. Inquiry form
  5. Login function

The appeal of Ruby is that you can develop web applications that include such functions more efficiently and easily than in other languages.

Services developed using Ruby

  • Airbnb
  • Kickstarter
  • Twitter
  • Hulu

These famous services are made using Ruby.

Summary

Ruby is a really good language for programming beginners because of its simple grammar and versatility. You can create any web application using Ruby on Rails.

If you are interested, download Ruby from the official Ruby website and start studying.
