Programming is to write a program, which is an instruction to a computer. For example, suppose you want a computer to add. If you want a human to do it, the instruction "calculate 1 + 1" is fine, but the computer cannot understand such an instruction. Therefore, what is needed is a programming language. A program is a program in which the work you want your computer to do is written in a language that your computer can understand. There are many types of programming languages, not just one. Since there are many types of work that can be done by a computer, there are various types of languages to make it easier to give instructions. Some languages　are optimized for one purpose and are not suitable for other purposes, while others are tuned to make it easier to direct different tasks. Moreover, it is not easy to bring one language to a level that can be used at work. In short, it's worth considering carefully which language to learn. This website provides useful information for beginners to choose a language to learn, such as the characteristics of each programming language, their areas of expertise, and the difficulty of learning, for those who are about to learn a programming language. Please read each article to understand the characteristics of each language and refer to it.
